# HDPlus
A mod to allow you to change resolution in the in-game settings at runtime. Supports ultra-wide resolutions (21:9).  v80+

# Info

A simple mod to allow changing resolution in-game, with no other graphical settings like currently existing mods.

Resolutions can be changed via the extra setting in the in-game settings menu.
Ultrawide can be enabled via the extra setting in the in-game settings menu.
