# HDPlus
A mod to allow you to change resolution in the in-game settings at runtime. Supports ultra-wide resolutions. v80+

# Info

A simple mod to allow changing resolution in-game, with no other graphical settings like currently existing mods.

Resolutions can be changed via the extra settings in the in-game settings menu.

The mod have ultra-wide support too that needs to be enabled in the configs. Simply set "Enabled = false" to "true" under [Ultrawide]
You have to run the game once for the config file to get generated.
