# HDPlus
A mod to allow you to change resolution in the in game settings at runtime. Supports v80+

# Info

A simple mod to allow changing resolution in-game, with no other graphical settings like currently existing mods.
It has a config file to change resoulutions, but in the in-game settings under the "Graphics" tab there is a button to change it to preset values (vanilla, 720p, 1080p, 1440p, 2160p), so no need to edit config at all.
